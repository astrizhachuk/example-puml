```bsl
// СвойстваРегламентногоЗадания возвращает редактируемые свойства регламентного задания.
// 
// Возвращаемое значение:
//   - Структура - описание:
//      * Использование - Булево - признак использования регламентного задания;
//      * Ключ - Строка - ключ регламентного задания;
//      * Параметры - Массив - параметры регламентного задания;
//      * Расписание - РасписаниеРегламентногоЗадания - расписание регламентного задания;
//      * ИмяПользователя - Строка - имя пользователя, под которым будет выполняться регламентное задание;
//
Функция СвойстваРегламентногоЗадания() Экспорт
```

### СохранитьНастройкуФормыРасчетаПриЗакрытии

```bsl
// СохранитьНастройкуФормыРасчетаПриЗакрытии выполняет сохранение параметров расчета для указанной формы
// при ее закрытии. Параметры расчета и их значения берутся из реквизитов формы. Имя реквизита формы должно
// совпадать с ключом параметров расчета целевых товарных запасов
// (см. ЦелевыеТоварныеЗапасыКлиентСервер.ПараметрыРасчета).
//
// Параметры:
//  Форма - УправляемаяФорма - форма, для которой выполняется сохранение параметров расчета;
//
Процедура СохранитьНастройкуФормыРасчетаПриЗакрытии(Знач Форма) Экспорт
```
