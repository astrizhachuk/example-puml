### ПолучитьДетализациюРасчета

```bsl
// ПолучитьДетализациюРасчета считывает ранее сохраненную детальную информацию о расчете
// целевого товарного запаса в т.ч. и при ошибках расчета.
//
// Параметры:
//  Ключ - РегистрСведенийКлючЗаписи - ключ записи с детализацией

// 
// Возвращаемое значение:
//   - Неопределено - запись не была найдена (прочитана);
//   - Структура - прочитанная запись:
//  * ПараметрыРасчета -  ХранилищеЗначения - структура с параметрами формирования целевого товарного запаса;
//  * РасчетныеДанные - ХранилищеЗначения - массив структур с данными для расчета целевых товарных запасов;
//
Функция ПолучитьДетализациюРасчета(Знач Ключ) Экспорт
```

### Рассчитать

```bsl
// Рассчитать возвращает результат расчета целевых товарных запасов.
//
// Параметры:
//  Параметры - Структура - см. ЦелевыеТоварныеЗапасыКлиентСервер.ПараметрыРасчета;
// 
// Возвращаемое значение:
//   - ТаблицаЗначений - рассчитанные целевые товарные запасы:
//  * ПодразделениеРасчета - СправочникСсылка.Подразделения - подразделение для которого производится расчет;
//  * Подразделение - СправочникСсылка.Подразделения - подразделение по которому производится расчет;
//  * НоменклатураУровень3 - СправочникСсылка.Номенклатура - каталог номенклатуры (3 уровень вложенности);
//  * НоменклатураУровень4 - СправочникСсылка.Номенклатура - каталог номенклатуры (4 уровень вложенности);
//  * Номенклатура - СправочникСсылка.Номенклатура - номенклатура;
//  * Код - Строка - код номенклатуры;
//  * ОсновнойПоставщик - СправочникСсылка.Контрагенты - основной поставщик;
//  * Отправитель - СправочникСсылка.ПТР_СкладскиеГруппы - отправитель по данным канала поставок;
//  * КроссДокинг - Булево - признак кросс-докинга по данным канала поставок;
//  * ПаллетныеОтгрузки - Булево - признак наличия паллетных отгрузок по данным канала поставок;
//  * Кратность - Число - коэффициент с единицы изменения номенклатуры по каналу поставок;
//  * ОсновнаяЕдиницаИзмерения - СправочникСсылка.ЕдиницыИзмерения - единица хранения остатков для номенклатуры
//                                  по каналу поставок или пустая ед. измерения при отсутствии данных;
//  * ТипОстатка - СправочникСсылка.БИТ_ТипыОстатков - срез последних типа остатка номенклатуры по подразделению;
//  * СреднееЗначениеПродажЧистое - Число - расчетное значение среднего значения продаж (чистое);
//  * СкользящаяСредняяПродаж - Число - расчетное значение скользящей средней продаж;
//  * КоэффициентСезонности - Число - коэффициент сезонности;
//  * СреднееЗначениеПродаж - Число - расчетное значение среднего значения продаж;
//  * СреднеквадратическоеОтклонение - Число - квадрат среднеквадратического отклонения
//                                              (корень извлекается постобработкой запроса);
//  * СтандартныйОбъемДефицитаЗапаса - Число - квадрат стандартного объема дефицита запаса
//                                              (корень извлекается постобработкой запроса);
//  * КоэффициентОбслуживания - Число - расчетное значение коэффициента обслуживания;
//  * ABCКласс - ПеречислениеСсылка.ABCКлассификация - ABC-классификация;
//  * КратностьРазовойПокупки - Число - расчетное значение кратности разовой покупки;
//  * ТекущийЦелевойТоварныйЗапас - Число - расчетное значение текущего целевого товарного запаса;
//  * ТекущийЦелевойТоварныйЗапасИКратность - Число - расчетное значение текущего
//                                                    целевого товарного запаса вместе с кратностью;
//  * ТекущаяЕмкость - Число - текущая емкость;
//  * ДатаИзмененияЦелевогоТоварногоЗапаса - Дата - дата последнего изменения целевого товарного запаса;
//  * АвторИзмененияЦелевогоТоварногоЗапаса - СправочникСсылка.Пользователи - автор последнего изменения
//                                                                               целевого товарного запаса;
//  * ЦенаЗакупки - Число - цена закупки товара;
//  * ЧастотаПросмотра - Число - расчетное значение частоты просмотра;
//  * ВремяДоставкиСМоментаЗаказа - Число - расчетное значение времени доставки с момента заказа товара;
//  * НеснижаемыйЦелевойТоварныйЗапас - Число - неснижаемый запас товара;
//  * УровеньСервисаПлан - Число - плановый уровень сервиса;
//  * УровеньСервисаФакт - Число - фактический уровень сервиса;
//  * СтраховойЗапас - Число - страховой запас;
//  * РасчетныйЦелевойТоварныйЗапас - Число - расчетный целевой товарный запас;
//  * РасчетнаяЕмкость - Число - расчетная емкость;
//  * ИзменениеЦелевогоТоварногоЗапаса - Число - расчетное значение изменения целевого товарного запаса;
//  * ПроверкаЕмкости - Число - расчетное значение проверки емкости;
//  * ИзменениеЕмкости - Число - расчетное значение изменения емкости;
//
Функция Рассчитать(Знач Параметры) Экспорт
```
