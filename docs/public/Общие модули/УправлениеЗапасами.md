### ЭтоРаспределительныйЦентр

```bsl
// ЭтоРаспределительныйЦентр возвращает результат проверки подразделения
// на принадлежность его распределительному центру (РЦ).
//
// Параметры:
//  Подразделение - СправочникСсылка.Подразделения - проверяемое подразделение;
// 
// Возвращаемое значение:
//   - Булево - Истина - это распределительный центр, иначе - Ложь; 
//
Функция ЭтоРаспределительныйЦентр(Знач Подразделение) Экспорт
```

### ПолучитьПодразделенияРаспределительныхЦентров

```bsl
// ПолучитьПодразделенияРаспределительныхЦентров возвращает все подразделения распределительных центров (РЦ).
// 
// Возвращаемое значение:
//   - Массив из СправочникСсылка.Подразделения - перечень подразделений распределительных центров (РЦ);
//
Функция ПолучитьПодразделенияРаспределительныхЦентров() Экспорт
```
