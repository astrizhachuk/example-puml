### РассчитатьДатуСдвига

```bsl
// РассчитатьДатуСдвига возвращает дату за вычетом сдвига периода, заданного в неделях.
//
// Параметры:
//  Дата - Дата - исходная дата за вычетом количества недель (сдвиг периода);
//  Недель - Число - количество недель, на которые будет уменьшена дата;
// 
// Возвращаемое значение:
//   - Дата - рассчитанная дата за вычетом сдвига периода;
//
Функция РассчитатьДатуСдвига(Знач Дата, Знач Недель) Экспорт
```

### РассчитатьПоказатель

```bsl
// РассчитатьПоказатель производит расчет изменений между значениями показателя в процентном соотношении.
//
// Параметры:
//  Показатели - СтрокаТаблицыЗначений, ДанныеФормыЭлементКоллекции - коллекция данных для расчета показателей;
//  Изменение - Строка - имя показателя, для которого рассчитывается изменение (результат расчета, %);
//  ИмяТекущего - Строка - имя показателя с текущим значением;
//  ИмяРасчетного - Строка - имя показателя с расчетным значением;
//
Процедура РассчитатьПоказатель(Показатели, Знач Изменение, Знач ИмяТекущего, Знач ИмяРасчетного) Экспорт
```

### РассчитатьЕмкость

```bsl
// РассчитатьЕмкость производит расчет емкости по показателям и способу округления результата расчета.
// При отсутствии способов округления при клиентском вызове вызывает исключение.
//
// Параметры:
//  Показатели - СтрокаТаблицыЗначений, ДанныеФормыЭлементКоллекции - коллекция с показателями, обязательные поля:
//  * РасчетныйЦелевойТоварныйЗапас - Число - расчетный целевой товарный запас;
//  * Кратность - Число - кратность единицы измерения;
//  * РасчетнаяЕмкость - Число - результат расчета емкости;
//  СпособОкругления - ПеречислениеСсылка.СпособыОкругленияЕмкости - способ округления при расчете;
//  СпособыОкругления - Неопределено, Структура - варианты округления, если Неопределено, то варианты определяются
//                  по значениям перечисления СпособыОкругленияЕмкости (используется для клиентского варианта вызова).
//                  При отсутствии способов округления при вызове с клиента - возникает исключение:
//  * ОкруглятьКБольшему - ПеречислениеСсылка.СпособыОкругленияЕмкости - значение перечисления
//                              для округления к большему;
//  * МатематическоеОкругление - ПеречислениеСсылка.СпособыОкругленияЕмкости - значение перечисления
//                              для математического округления;
//
Процедура РассчитатьЕмкость(Показатели, Знач СпособОкругления, Знач СпособыОкругления = Неопределено) Экспорт
```
